常见问题

OpenClaw Companion 是什么？
OpenClaw Companion 是 OpenClaw AI 机器人框架的桌面配置管理工具。它让你不用手动编辑 YAML 文件就能完成所有配置，包括模型选择、频道绑定、备份迁移等操作。
需要先安装 OpenClaw 吗？
不需要。Companion 内置了 OpenClaw 安装检测功能，首次启动会引导你安装。你只需要确保系统已安装 Node.js 22+。
支持哪些操作系统？
支持 Windows 10/11 (64-bit)、macOS (Intel 和 Apple Silicon)、Linux (Ubuntu、Debian、Fedora、Arch 等主流发行版)。
配置数据存在哪里？安全吗？
所有数据（模型配置、频道设置、对话记忆等）存储在你本地电脑的 ~/.openclaw/ 目录中，不会上传到任何服务器。你可以随时通过备份功能导出。
macOS 提示"无法验证开发者"怎么办？
右键点击应用 → 选择"打开" → 在弹窗中点击"打开"即可。这是因为应用未经 Apple 公证签名，不影响正常使用。
完全免费吗？
是的，OpenClaw Companion 完全免费，没有任何付费功能或内购。你需要自备大模型的 API Key（部分模型有免费额度）。
如何获取更新？
应用内置自动更新功能，有新版本时会弹窗提醒。你也可以在侧边栏手动点击"检查更新"。
